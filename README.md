# Time Buddy ✈️

世界中の都市の時間を一目で確認できる、シンプルで美しいタイムゾーン管理アプリです。

## 概要
「Time Buddy」は、東京、ロンドン、ニューヨークなど、主要都市の現在時刻をリアルタイムで表示します。
また、特定の時間を入力することで、他の都市がその時何時になるかを瞬時に計算する「タイムトラベル（変換）」機能も備えています。

## 主な機能
- **リアルタイム時計**: 世界8都市の現在時刻を表示（秒単位で更新）。
- **手動時刻変換**: 任意の都市の時間を変更すると、他のすべての都市の時間が自動的に変換されます。
- **日付変更機能**: 日付をクリックしてカレンダーから日を選ぶと、サマータイム（DST）を考慮した正確な時間をシミュレーションできます。
- **Live/Resumeモード**: 手動操作中は時計が止まり、ヘッダーの「Resume」ボタンでいつでもリアルタイムに戻れます。
- **レスポンシブデザイン**: PCでは2列表示、スマホでは見やすい1列表示に自動調整されます。

## 収録都市
- メルボルン (MEL)
- 東京 (HND)
- シンガポール (SIN)
- バンガロール (BLR)
- ドバイ (DXB)
- ポーランド (WAW)
- ロンドン (LHR)
- ニューヨーク (JFK)

## 使い方（Webページとして公開する方法）
このアプリをスマホや他のPCからアクセスできるようにするには、GitHub Pagesを利用するのが一番簡単です。

1. GitHubのリポジトリページを開きます: [https://github.com/ShoKaneshiro/time-zone-buddy](https://github.com/ShoKaneshiro/time-zone-buddy)
2. **Settings** (設定) タブをクリックします。
3. 左側のメニューから **Pages** を選択します。
4. **Build and deployment** の **Branch** を `None` から `main` に変更し、**Save** をクリックします。
5. 数分待つと、ページ上部に公開URLが表示されます（例: `https://ShoKaneshiro.github.io/time-zone-buddy/`）。

このURLにアクセスすれば、いつでもどこでもアプリを使えます！
