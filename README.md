A simple and beautiful timezone management app that lets you check the time in major cities at a glance.

## Overview
**Time Buddy** displays the real-time current time for key cities like Tokyo, London, and New York.
It is designed specifically with **common timezones for APAC Product Developers** in mind, helping you coordinate across global teams effortlessly.

It also features a "Time Travel (Converter)" mode‚Äîsimply input a time for any city, and it instantly calculates what time it will be in all other cities.

## Key Features
- **Real-time Clock**: Displays current time for 8 major cities (updates every second).
- **Manual Time Converter**: Change the time for any city, and all others update automatically.
- **Date Simulation**: Click the date to select a specific day from the calendar to simulate accurate times considering Daylight Saving Time (DST).
- **Live/Resume Mode**: The clock pauses during manual interaction. Use the "Resume" button in the header to return to real-time.
- **Responsive Design**: Automatically adjusts to a 2-column layout on desktop and a mobile-friendly 1-column layout on phones.
